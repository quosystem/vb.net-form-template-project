# report フォルダ
帳票ファイルを格納します。

- Crystal Reports などの帳票レイアウトファイル（`.rpt`）をここに置きます。
- コードからは相対パスで参照できるようにします。

---

## 共通設定（アプリ不具合対応）

Crystal Reportsにおける画像の描画不具合を防ぐため、以下の設定を必ず行ってください。

1. **Crystal Reportsのレポートオプション**  
   - 「画像の元のカラーデプスを保持」に **チェックを入れる**  
   - 対象: Windows 11 で帳票出力時に画像にノイズが入る現象への対応  
   - 詳細は以下Slackスレッドを参照:  
     [Slackリンク](https://quocard1987.slack.com/archives/G01GB7952PK/p1755251551317879)
